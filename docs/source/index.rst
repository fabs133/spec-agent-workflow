Spec-Agent Workflow System
==========================

A **Spec-Pattern Multi-Agent Architecture** for structured text extraction.

The system loads text files, extracts structured items via an LLM, and writes
results -- validating every step with pure specification functions and recording
a complete execution trace to a SQLite database.
